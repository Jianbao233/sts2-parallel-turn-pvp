# 代码结构与工作交接

## 目录分层
- `src/ParallelTurnPvp/Bootstrap`
  - 调试房间入口、custom run 接管、固定测试内容发放
- `src/ParallelTurnPvp/Core`
  - PvP runtime、消息、resolver、frontline helper、white list、direct IP 兼容
- `src/ParallelTurnPvp/Models`
  - Debug modifier、自定义卡牌、药水、遗物
- `src/ParallelTurnPvp/Patches`
  - Harmony 补丁，负责菜单入口、战斗目标、同步护栏、视觉重排、回合跟踪
- `src/ParallelTurnPvp/Ui`
  - PvP 专用 overlay 和后续 UI
- `analysis`
  - 架构、设计、里程碑、变更、交接文档

## 当前关键文件
- `Bootstrap/ParallelTurnPvpArenaBootstrap.cs`
  - 进入调试模式、配置固定测试局
- `Core/PvpModels.cs`
  - `PvpMatchRuntime`、`PvpRoundState`、有限意图 runtime 的主宿主
- `Core/PvpNetBridge.cs`
  - 当前 authoritative 消息桥；注意：当前只更新 runtime，不再写回原版 live combat
- `Core/PvpRoundResolver.cs`
  - 目前仍偏结果摘要器，是后续延迟结算改造的核心切入口
- `Core/ParallelTurnFrontlineHelper.cs`
  - 前线优先目标、前线/本体关系、frontline 查询
- `Patches/CombatPatches.cs`
  - 战斗主补丁：目标规则、回合切换、overlay 挂载、视觉层整理、胜负流程
- `Patches/RoundTrackingPatches.cs`
  - 动作记录、锁回合、结束回合撤销拦截
- `Ui/ParallelTurnIntentOverlay.cs`
  - 首版有限意图 overlay
